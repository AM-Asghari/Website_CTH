# Website voor CTH

CHEATSHEET!

  1. Eerst moet je een github repository aanmaken op github waar jij je bestand wilt uploaden.
  2. Als je nog geen map hebt: Open je terminal. In de terminal type je mkdir dan maak je een folder aan in die directory.
  3. Ga naar de map door cd (naam_map) te gebruiken en cd.. om terug te gaan
  4. git init dit maakt een .git folder aan in de map
  5. git status om te kijken of er bestanden klaar staan die je wilt comitten.
  6. git add (naambestand) of git add . dit voegt alle bestanden in een keer toe.
  7. git commit -m "Beschrijving van wat je hebt gewijzigd"
  8. git branch -M main
  9. git remote add origin link van je github repository
  10. git push -u origin main
